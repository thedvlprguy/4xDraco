## 📋 Requirements

- Linux/Unix-based system
- Root/Sudo privileges
- Git
- Python 3.x
- PHP
- Curl

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jhwach/4xDraco.git
   ```

2. Navigate to the directory:
   ```bash
   cd 4xDraco
   ```

3. Make the script executable:
   ```bash
   chmod +x 4xDraco.sh
   ```

4. Run the script:
   ```bash
   ./4xDraco.sh
   ```
   Or with sudo privileges:
   ```bash
   sudo ./4xDraco.sh
   ```

